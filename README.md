#  Expense Tracker

A personal expense tracker built with Python.

##  About The Project
Track your daily expenses by category, view spending summaries, and manage your personal budget — all from the terminal.

##  Features
- Add and delete expenses
- Categorize spending (Food, Transport, Shopping etc.)
- Monthly summary and totals
- Budget limit with overspending alerts
- Most expensive category finder
- Save and load expenses from a file

##  Built With
- Python 3
